# Map Tracking

Chrome extension that adds a route map overlay to UPS tracking pages.

## Demo

<video src="./demo/ups-package-route.mp4" controls muted playsinline></video>

[MP4 file](./demo/ups-package-route.mp4)

## Use

```bash
npm install
npm run build
```

Open `chrome://extensions`, enable Developer mode, load this folder as an unpacked extension, then open a UPS tracking page and click `Route Map`.
