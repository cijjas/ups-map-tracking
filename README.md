# Map Tracking

Chrome extension that adds a route map overlay to UPS tracking pages.

https://github.com/user-attachments/assets/ed182fc9-6cc2-4ef0-a2d8-344461a5719e

[MP4 file](https://github.com/user-attachments/assets/ed182fc9-6cc2-4ef0-a2d8-344461a5719e)

## Use

```bash
npm install
npm run build
```

Open `chrome://extensions`, enable Developer mode, load this folder as an unpacked extension, then open a UPS tracking page and click `Route Map`.
